# Aurora Financial — Security & Compliance

## Certifications
Aurora is SOC 2 Type II certified, ISO 27001 certified, and PCI-DSS Level 1 compliant.

## Data protection
- All data is encrypted at rest (AES-256) and in transit (TLS 1.3).
- Customer data is logically isolated per tenant.
- Secrets are managed via a hardware security module (HSM).

## Access control
- Role-based access control (RBAC) across all internal systems.
- Mandatory multi-factor authentication (MFA) for all employees.
- Production access requires just-in-time approval and is fully audited.

## Incident response
Aurora maintains a 24/7 security operations center. Critical incidents are triaged
within 15 minutes, and affected customers are notified within 4 hours per contractual
SLA.
