# Aurora Financial — Products

## Aurora Core
A cloud-native core banking platform handling accounts, ledgers, payments, and
settlement. Exposes REST and event-streaming (Kafka) APIs. Supports real-time
transaction processing with 99.99% uptime SLA.

## Aurora Wealth
A wealth-management suite for portfolio construction, rebalancing, and reporting.
Includes a robo-advisor module and risk-profiling questionnaires.

## Aurora Shield
A fraud-detection and AML (anti-money-laundering) product using rules plus machine-
learning models to flag suspicious activity in real time.

## Pricing model
Aurora charges a base platform fee plus per-transaction pricing. Wealth and Shield are
add-on modules billed separately. Enterprise customers get volume discounts and a
dedicated support team.
